Användarmanual för Befolkningsstatistik.py:

Syfte: 
Detta projekt har jag skapat för att demonstrera hur man kan använda SCB's API i Python-programmering. 
Programmet har en enkel user interface där man kan välja en eller flera regioner för att hämta aktuell befolkningsstatistik från SCB via deras API.

Instruktioner:
Innan filen öppnas så krävs det att python 3 är installerat 
samt modulerna pandas och matplotlib

Öppna sedan filen och följ nedan instruktioner:

1. Skriv in regionkoden som visas till vänster om regionen och tryck på enter för att lägga till regionen i listan.
2. Tryck på enter om ytterligare region ska läggas till eller skriv in "q" och sedan enter för att visa befolkningsutveckling för angiven region
3. Upprepa punkt 1 och 2 för att lägga till ytterligare regioner tills du är nöjd och tryck sedan på "q" för att visa och jämföra befolkningsutveckling mellan regionerna.
